# 关于选板

假设敌方车辆半径25cm，两转每秒的情况下，瞄准中心和最近装甲板两种情况的子弹飞行时间内敌方小陀螺转动角度差距不超过10度，故可先计算瞄准distance减去radius距离下的子弹飞行时间做预判（几十毫秒，敌方旋转几十至几百度），再进行装甲板选择（避免实际击打和瞄准错开）。

因为选择击打此时最面对的装甲板进行击打，锁定这块装甲板后进行迭代飞行时间计算（不在每次迭代时重新选板，避免迭代时反复横跳无法收敛）。如果出现“迭代后期锁定装甲板预判旋转至背对”的极端情况（基本不可能，除非观测器发散出现异常值或者对面真的能转到每秒十圈以上），直接放弃该帧瞄准（而不是重新锁定别的装甲板，因为这种状况已经违背物理常识了）。

对于将近静止状态下斜45度附近的目标，为避免观测器噪声导致的帧间选板反复，对非锁定装甲板面对角度相对锁定装甲板面对角度差的绝对值设置一个死区（即当非锁定装甲板明显更加面对时才进行切板）。
