# 能量机关追踪节点

订阅识别到的扇叶和云台状态，发布瞄准命令。

需要依赖操作手手动单发。

## 关于小能量机关

使用重投影误差约束的因子图优化解决。详细思路可以参考armor_tracker的README。每一个扇叶连接了五个重投影约束因子（五个关键点），所有新识别的扇叶经过匹配后通过降自由度因子连接到风车状态上（三维位置和横滚，即默认风车正对）。

## 关于大能量机关

是小能量机关的超集。使用ceres拟合变速旋转。由于将转速写死在约束条件里，使用视频离线测试时会出现拟合异常的情况（因为视频播放速率难以控制）。建议直接上车对着风车调试。实际开大符还是很稳的。

为了实现大小符状态类预测方法的类型擦除和链式调用，在大符获得能量机关状态的方法中使用了超级无敌霸霸龙闪电雷霆宇宙无敌曼波哇嗷的lamda表达式，死18,446,744,073,709,551,615个道士都没用的那种，擅自阅读可能导致不可挽回的脑损伤。
